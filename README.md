### 코드 제출 방법
1. 자신의 로컬 저장소에 원격 레포지토리를 pull
2. 해당하는 주차의 폴더 안에 자신의 백준 아이디를 이름으로 하는 폴더를 생성 
ex) 01-BruteForce-and-Backtracking/[name]
3. 폴더 안에 자신의 정답 코드를 작성
- 코드 파일의 제목은 문제의 알파벳 번호-문제 번호
ex) 01-BruteForce-and-Backtracking/ccoco/A-2309.cpp
4. 작성이 끝났다면 master branch로 push

## ✍️ 풀이 작성 가이드 (PR에 포함 권장)

- **접근 아이디어**: 왜 이 자료구조/알고리즘인가?
- **시간/공간 복잡도**: Big-O로 한 줄
- **실수 포인트**: 구현에서 헷갈렸던 조건/엣지케이스

예시(주석 or PR 본문):
- Idea: 스택에 여는 괄호 push, 닫는 괄호면 top 확인
- Time: O(N), Space: O(N)
- Pitfall: 빈 스택에서 pop 시도, 문자열 끝에서 스택이 비었는지 확인
